TEMA 2

GRUNDLÆGGENDE WEB

02.01.02 WEBSITE

Grundlæggende Web var vores første tema, og var derfor også første gang vi prøvede kræfter med HTML, CSS og alt der hører til, inde i programmet VS Code. Derudover lærte vi om responsivt design, og vigtigheden af, at brugeren kan benytte sitet på alle skærme.
I dette tema begyndte vi også med at lære mange andre basis-koncepter indenfor multimediedesign som gestaltlove og designkonventioner, som vi i denne opgave brugte til at skabe vores første brugergrænseflade.

På Computersitet benyttede jeg mig af min nye viden om grids i css, media-queries og layoutdiagrammer. På denne måde fik jeg placeret det indhold vi fik udleveret ifølge layoutdiagrammerne, og stylet siden så den fik et sammenhængde look.
